# 基于视觉伺服的随动式螺钉拧紧机器人代码库分析

## 项目概述

本项目是一个基于Aubo机械臂的视觉伺服控制系统，集成了RealSense深度相机、夹爪控制、手眼标定和工具标定等功能，用于实现视觉引导的机器人抓取和操作任务。

---

## 主要程序文件及功能说明

### 1. **Aubo_Robot.py** - 核心机器人控制类

**功能**：集成机器人控制、相机、夹爪等功能的统一接口类

**主要特性**：
- 继承自 `Auboi5Robot`，封装了机器人控制功能
- 集成RealSense相机数据获取
- 集成夹爪控制（JawController）
- 提供坐标变换工具（RPY与旋转矩阵转换）
- 实现抓取和推动等高级操作

**使用方法**：
```python
from Aubo_Robot import Aubo_Robot

# 初始化系统
Aubo_Robot.initialize()

# 创建机器人实例（可选择是否连接相机和夹爪）
robot = Aubo_Robot(
    robot_host_ip="192.168.65.100",  # 机器人IP地址
    robot_port=8899,                  # 端口
    is_connect=True,                  # 是否连接机器人
    is_use_jaw=True,                  # 是否使用夹爪
    is_use_camera=True                # 是否使用相机
)

# 回到初始位置
robot.go_home()

# 平面抓取（适用于水平面上的物体）
robot.plane_grasp(position=[0, 0.4, 0.3], yaw=0)

# 通用抓取（可指定RPY角度）
robot.grasp(position=[0.4, 0, 0.3], rpy=[-np.pi, 0, 0])

# 推动操作
robot.plane_push(position=[0.2, 0.2, 0.1], move_orientation=0, length=0.1)

# 获取相机数据
color_img, depth_img = robot.get_camera_data()

# 控制夹爪
robot.open_gripper()   # 打开夹爪
robot.close_gripper()  # 闭合夹爪
```

**关键参数**：
- `workspace_limits`: 工作空间限制 [[x_min, x_max], [y_min, y_max], [z_min, z_max]]
- `home_joint_config`: 初始关节角度配置
- `cam_pose`: 相机相对于法兰的位姿（从camera_pose.txt加载）
- `tool_pose`: 工具相对于法兰的位姿（从tool_calibration_result.txt加载）

### 2. **demo.py** - 交互式抓取演示程序

**功能**：通过点击相机图像中的目标点，自动计算并执行抓取操作

**使用方法**：
```python
python demo.py
```

**操作流程**：
1. 程序启动后，机器人回到初始位置
2. 显示相机彩色图像和深度图像
3. 在彩色图像窗口中点击目标物体
4. 程序自动计算目标在机器人基坐标系下的位置
5. 执行抓取操作
6. 按 'c' 键退出程序

**关键技术**：
- 相机坐标系到机器人基坐标系的坐标变换
- 使用深度信息计算3D坐标
- 考虑工具坐标系偏移

### 3. **data_collect.py** - 手眼标定数据采集

**功能**：采集手眼标定所需的图像和机器人位姿数据

**使用方法**：
```python
python data_collect.py
```

**操作步骤**：
1. 将标定板放置在机器人工作空间内
2. 运行程序，机器人回到初始位置
3. 移动机器人到不同位置和姿态，使标定板在相机视野内
4. 按 's' 键保存当前图像和机器人位姿
5. 重复步骤3-4，至少采集20-30组数据
6. 按 ESC 退出

**输出文件**：
- `./collect_data/images0.jpg, images1.jpg, ...` - 采集的图像
- `./collect_data/poses.txt` - 对应的机器人位姿（每行格式：x,y,z,rx,ry,rz）

**注意事项**：
- 确保标定板清晰可见
- 机器人姿态要有足够的变化
- 建议采集30组左右的数据

### 4. **hand_eye_calibrate.py** - 手眼标定程序

**功能**：计算相机坐标系相对于机器人末端法兰坐标系的变换矩阵（眼在手上）

**使用方法**：
```python
python hand_eye_calibrate.py
```

**标定原理**：
- 使用棋盘格标定板进行相机标定
- 通过多组机器人位姿和对应的标定板图像
- 使用TSAI方法求解手眼标定方程：A₂⁻¹·A₁·X = X·B₂·B₁⁻¹

**配置参数**：
```python
iamges_path = "./collect_data"        # 标定图像路径
arm_pose_file = "./collect_data/poses.txt"  # 机器人位姿文件
N = 30                                 # 采集图像数量
XX = 11                                # 标定板长度方向角点数
YY = 8                                 # 标定板宽度方向角点数
L = 0.015                              # 标定板格子尺寸（米）
```

**输出结果**：
- 旋转矩阵 R (3×3)
- 平移向量 t (3×1)
- 结果保存到 `camera_pose.txt`（4×4齐次变换矩阵格式）

**标定流程**：
1. 运行 `data_collect.py` 采集数据
2. 运行 `hand_eye_calibrate.py` 进行标定
3. 检查标定结果是否合理
4. 将结果保存到 `camera_pose.txt`

### 5. **tool_calibrate.py** - 工具坐标系标定

**功能**：标定工具（如夹爪、拧紧工具）相对于机器人末端法兰的位姿

**使用方法**：
```python
python tool_calibrate.py
```

**标定原理**：
- **平移标定（TCP）**：通过机器人在不同姿态下触探同一固定点，计算工具尖端位置
- **旋转标定（TCF）**：通过工具在不同方向的指向确定坐标系

**数据准备**：
- 准备 `tool_data.csv` 文件，包含至少6组机器人位姿数据
- 格式：每行 [x, y, z, rx, ry, rz]（位置单位：米，角度单位：度）

**标定步骤**：
1. **平移标定**（使用前4个点）：
   - 将工具尖端指向同一固定点
   - 记录4个不同的机器人位姿
   
2. **旋转标定**（使用后3个点）：
   - 工具指向不同方向
   - 记录3个机器人位姿

3. **验证**（使用剩余点）：
   - 验证标定精度

**输出结果**：
- 工具坐标系变换矩阵（4×4齐次变换矩阵）
- 保存到 `tool_calibration_result.txt`

**示例数据格式**（tool_data.csv）：
```
0.4, 0.0, 0.3, -180, 0, 0
0.4, 0.1, 0.3, -180, 0, 0
0.3, 0.0, 0.3, -180, 0, 0
0.4, 0.0, 0.2, -180, 0, 0
0.4, 0.0, 0.3, -180, 0, 45
0.4, 0.0, 0.3, -180, 0, 90
```

### 8. **jaw_control.py** - 夹爪控制

**功能**：通过串口控制夹爪的开关

**使用方法**：
```python
from jaw_control import JawController

# 创建夹爪控制器
jaw = JawController()

# 打开串口（默认 /dev/ttyACM0，Windows下可能是 COM3）
jaw.port_open(port='COM3', baudrate=115200)

# 控制夹爪
jaw.jaw_control(1)  # 闭合
jaw.jaw_control(0)  # 张开

# 复位（闭合后张开）
jaw.reset()

# 关闭串口
jaw.port_close()
```

**串口配置**：
- 波特率：115200
- 数据位：8
- 停止位：1
- 校验位：无

**Windows系统**：串口名称通常是 `COM1`, `COM2`, `COM3` 等

### 9. **move_joint.py** - 关节空间运动示例

**功能**：演示如何控制机器人进行关节空间运动

**使用方法**：
```python
python move_joint.py
```

**主要功能**：
- 连接机器人
- 设置运动参数（速度、加速度）
- 执行关节空间运动
- 获取当前关节角度

**关键代码**：
```python
# 设置关节最大速度和加速度
joint_maxvelc = (2.596177/15, 2.596177/15, ...)
joint_maxacc = (17.308779/5, 17.308779/5, ...)
robot.set_joint_maxacc(joint_maxacc)
robot.set_joint_maxvelc(joint_maxvelc)

# 移动到指定关节角度（弧度）
joint_radian = (-90/180.0*pi, 0, -90/180.0*pi, 0, -90/180.0*pi, 0)
robot.move_joint(joint_radian, True)
```

## 配置文件说明

### camera_pose.txt
相机相对于机器人末端法兰的4×4齐次变换矩阵
```
R11 R12 R13 tx
R21 R22 R23 ty
R31 R32 R33 tz
0   0   0   1
```

### tool_calibration_result.txt
工具相对于机器人末端法兰的4×4齐次变换矩阵
```
R11 R12 R13 tx
R21 R22 R23 ty
R31 R32 R33 tz
0   0   0   1
```

### camera_depth_scale.txt
深度图像的缩放因子（通常为0.001，表示1mm）

## 注意事项

1. **网络连接**：确保机器人IP地址正确（默认：192.168.65.100:8899）

2. **相机连接**：确保RealSense相机正确连接并安装驱动

3. **串口配置**：夹爪控制需要正确的串口配置（Windows: COM口，Linux: /dev/ttyACM0）

4. **工作空间限制**：注意设置合理的工作空间限制，避免机器人碰撞

5. **标定质量**：标定精度直接影响系统性能，确保标定数据质量

6. **安全操作**：操作机器人时注意安全，建议先低速测试

## 常见问题

1. **机器人连接失败**：检查IP地址和端口，确保机器人控制器已启动
2. **相机无法获取数据**：检查相机连接和驱动安装
3. **夹爪无响应**：检查串口配置和硬件连接
4. **标定精度低**：增加标定数据量，确保数据质量
5. **坐标变换错误**：检查标定文件是否正确加载
